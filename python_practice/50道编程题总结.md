## 四、能力覆盖评价

- **语法基础**：变量、数据类型、运算符、格式化输出 → 全覆盖
- **控制流程**：`if` 多分支、`while` 循环、`for` 双层嵌套 → 高频练习
- **数据结构**：列表的增删改查、二维矩阵、索引操作 → 手写排序和数组操作
- **算法思想**：递归、贪心、筛选法、逆向倒推、约瑟夫环 → 经典固定题型
- **工程意识**：第 50 题串起 `input` 循环 + 数据组装 + `with open` 写文件，是唯一一道带 I/O 的综合题

## 五、学习建议

1. **手写不依赖**：每题先自己写，写完再对照参考，建立肌肉记忆
2. **举一反三**：同一算法题尝试用不同方法解（如斐波那契：递归 / 迭代 / 通项公式）
3. **补齐短板**：50 题中文件 I/O 仅 1 题、面向对象为 0，后续可在项目中加强
4. **分类复习**：按本文档分类做模块化抽查，随机抽一类中的题重新手写
